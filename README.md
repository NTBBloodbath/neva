# neva
a neovim version manager


## Install

```bash
# download script
mkdir -p $HOME/.neva/bin
curl -L -o $HOME/.neva/bin/neva https://github.com/shohi/neva/raw/main/neva

# add to $PATH
export PATH="$HOME/.neva/bin":$PATH
```

Note: require `lua 5.3+` installed

## Usage

```bash
$> neva --help

```
